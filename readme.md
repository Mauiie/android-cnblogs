#博客园APP

非官方客户端，目前版本尚未发布。有兴趣的可以fork提交你的代码哦，有任何提议或者想法也可以提提issue。

**导入项目**

- `git clone 'https://git.oschina.net/rae/android-rae-core.git'`
- `git checkout dev`
- 把lib 目录导入到当前项目中
- 最后在sdk目录下的`build.gradle` 中将下面的：` compile 'com.rae.core:lib:1.0.0'` 改成`compile project(':lib')`
